# Home Manager Module Caveats

## Non-available settings

The following settings can't be modified because they are out of the file
namida_settings.json, which is the file namida-nix modifies, however you can
toggle them in Namida settings.

### namida_settings_player.json

1. `displayRemainingDurInsteadOfTotal`
2. `displayActualPositionWhenSeeking`
